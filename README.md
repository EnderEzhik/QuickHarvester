Мод для Minecraft 1.21.1 Forge упрощающий выращивание сельскохозяйственных культур.

Доступные функции:
- Медленный сбор урожая - с помощью зажатой пкм постепенно собирает весь созревший урожай
- Быстрый урожай - с зажатым шифтом правой кнопкой мыши собирается сразу весь урожай

P.S. Обрабатываем растения в пределах одной цельной грядки.

Будущие функции:
- Автоматическое засеивание грядки сразу после сбора урожая
